# 設計思想

## コンセプト

**「静かな叫びのための牧場」**

陰キャ同士が自然に交流できる場を提供する。承認欲求を排除し、存在証明としての言葉を残す空間。

## 世界観

### メタファー

- **牧場**：全ユーザーが集まる唯一の空間
- **眼鏡アバター**：ユーザーの分身、放牧されている
- **厩舎**：個人の蓄積（消えない）
- **掲示板**：厩舎の写像（7日で消える）
- **柵に貼られた紙**：投稿の表現形式

### 視覚イメージ

- セザンヌ調の写実的な風景
- 曇り空、風に揺れる紙
- ハイラル平原のような儚さ
- アニメ調ではない

### トーン

- 「車を持たず、どこにも行けない」閉塞感
- 若いリバー・フェニックスのような鬱憤
- 海の近くにない静けさ
- 不条理ギャグ（「グラスにグラスを食わせる」）

## 設計原則

### 1. 承認欲求の排除

- いいね、フォロワー数、ランキングなし
- 閲覧数は非公開（本人にもデフォルト非表示）
- 評価機能なし

### 2. 蓄積型

- 厩舎の投稿は消えない
- 「書いたものが積み重なる」ことが満足になる
- タイムラインの流れではなく、アーカイブ

### 3. 偶発性の再現

- 擬似ではなく、実際の時間経過で交流が発生
- ガンビット + 乱数で「必ずではない交流」
- 趣味ベクトルで交流確率を自然に高める

### 4. 匿名性と自然消滅

- 登録不要、UUID + 仮名のみ
- 90日アクセスなしで眼鏡が消滅
- 国籍・学歴・職業などバイアス情報を排除

### 5. 平等性

- おすすめアルゴリズムなし
- タグ・分類なし
- 掲示板は純粋時系列のみ

## ユーザーができること

### 交流以外

- 映画感想（評価点なし）
- 読書感想
- 夢日記
- 妄想小説
- 思考の披露
- 存在証明ログ

### 禁止

- 写真・動画・音楽のアップロード
- 学歴・国籍・収入・職業の公開

## スクリーニング（陰キャメガネチェック）

初回リネーム時に実施。10問の2択、全問正解必須。

目的：体験のトーンを揃える儀式

特徴：
- 遊び心重視、心理的負担をかけない
- 道徳的評価はしない
- 失敗しても尊重的な文面
- 24時間ロック後に再挑戦可能
